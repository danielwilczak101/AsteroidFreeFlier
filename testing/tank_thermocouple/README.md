## Wiring
CS - Any digital pin (default GPIO 17, board pin 11)
SCK - SCLK (GPIO 11)
SDO - MISO (GPIO 9)
SDI - MOSO (GPIO 10)
Vin - 5V or 3V
GND - Ground
