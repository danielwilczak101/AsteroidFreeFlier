## Wiring
CS - Any digital pin (default GPIO 17, board pin 11)\n
SCK - SCLK (GPIO 11)\n
SDO - MISO (GPIO 9)\n
SDI - MOSO (GPIO 10)\n
Vin - 5V or 3V\n
GND - Ground\n
